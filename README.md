# GCal Extractor

## Install
```
npm install
npm start
```

## Dependencies

* gcalcli `pip install gcalcli`
* parsedatetime `pip install parsedatetime`

## Example
```
> curl localhost:9679/car.tsv

date  km  gas price
2014-11-04  407.5 38  46
2014-12-12  578 34.26 34.26
2015-01-30  594 36.87 39.41
```
